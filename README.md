# rda_ros
ROS Wrapper of RDA planner


## Environment

- Ubuntu 20.04
- ROS Noetic
- Python >= 3.8


## Preqrequisite

- [RDA_planner](https://github.com/hanruihua/RDA_planner)

```
git clone https://github.com/hanruihua/RDA_planner.git
cd RDA_planner
pip install -e .
```


## Build

```
cd catkin_ws
catkin_make 
```
