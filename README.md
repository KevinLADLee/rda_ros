# rda_ros

This is the ROS Wrapper of the [RDA planner](https://github.com/hanruihua/RDA-planner)

## Preqrequisite

Please install the RDA planner first.

- Python >= 3.8
- ROS Noetic
- [RDA_planner](https://github.com/hanruihua/RDA_planner)
